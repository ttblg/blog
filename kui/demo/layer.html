Layer核心类
基本参数
参数 类型 说明
id string 弹层ID，有唯一性
type string 弹层类型
head str/obj 层头部，数据格式：{title:'头部标题'，}
content string 内容区
footer string 脚注
skin string 自定义层样式(皮肤)名
size array 层尺寸，如：['300px','250px']或['80%','50%']
style str/obj 自定义CSS样式，如：{color:'#F00'}，或'color:#F00'
anim str/arr 入场效果，如：left/right/top/bottom/fade/zoom，也可以输入自定义效果，格式如：[{初始CSS},{入场动画},{出场动画},time]
autoshow boolean 是否自动显示，如果否，则需要用obj.show()方法去显示
shade boo/css 是否显示遮罩层，true或定义遮罩层的css样式
shadeclose boolean 点击遮罩层是否关闭
timeout number 秒，定时关闭遮罩层
scroller false 开始JS滚动，依赖Scroller.js
onshow function 当显示后回调函数
onclose function 当关闭后回调函数
方法
方法名 说明
$.layer.show(layer) 显示层，layer为层对象，或层ID
$.layer.close(layer,call,true) 关闭层，当layer为All时，可关闭所有层。同时call为层类型时，可以关闭同类型的所有层。第三个参数为关闭后是否删除Dom
事件
事件 说明
show 显示后触发
close 关闭后触发
$.alert类
基本参数
参数 类型 说明
content string 内容，必填
title string 标题，可不填
button Array 按扭，如['取消','确定']
callback Function 回调，可不填

$.alert(content,title,button,callback)，content为必填项，其他三个参数可以不填，也不讲究顺序。

$.actions(button,option)
button参数
参数 类型 说明
text string 按扭文字
label boolean 是否为Label，为true时，点击不隐藏，且内容左对齐
onClick fun/obj/none 点击事件，可以为Function，或连接{href:'网址',target:'_blank'}
className string 按扭样式，可以填写base中的状态背景，或状态文字样式小技巧：如果className带有unHide则点击不关闭。

$.actions(button,option)
button为按扭组，如[{text:'用户首页'},{text:'用户资料'},{},{text:'关闭'}]其中{}为分隔线
option继承$.layer核心参数所有属性

$.toast(option,icon)
参数
参数 类型 说明
option str/obj 文字内容，或核心类参数
icon string 图标，参考：icon
