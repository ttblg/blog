数据缓存
本缓存功能是通过重新封装localStorage实现，实现可以直接存取JSON、Array数据以及设置过期时间。

kui.cache(name,value,time)

参数 类型 说明
name string 必填
title str/json/arr 内容，为空则输出名称所对应的缓存值。
time Number 有效期（小时），默认6小时

kui.removeCache(name) 获取指定缓存

删除缓存，如name为空，则清除所有用kui.cache写入的缓存
Cooke操作
  • kui.addCookie(name, value, day, path)

    day:缓存天数，默认30天

    path:路径，默认针对跟目录

  • kui.getCookie(name) 读取Cookie
  • kui.delCookie(name) 删除Cookie
